Ration Shop Management

The Ration Shop Management application is developed using Java Swing for managing the daily availability of the stocks and providing it for the customers in an economically efficient process. So the main objective of our application is to keep track of the available stocks from the start of the day which includes the processes like deduction of the count of the sold-out stocks and reporting the left out stocks the next day.This application also consists of authenticating the (providing the customer's identity) based on the card number.The customer should be able to view their monthly purchase and also the billing details.

sPark

The developed software can recognise licence plates on cars entering campus and determine whether or not they are registered. The free parking spaces as well as the number plates are detected with the help of the DL models trained using the Yolov7 algorithm after collecting the required datasets from various sources. The free parking spaces that are detected with the help of these models are automatically updated in the database. Additionally, we check to see if the cars assigned parking spots are actually parked in them.

Robber Detection using YOLOV7

The model is built using the YOLOV7 deep learning algorithm, and it is trained using a custom dataset that has been collected from various sources. The created model helps banks identify robbers. The model can be used in banks and ATM systems that have any type of CCTV installed, and the video footage can be automatically analysed using openCV.
